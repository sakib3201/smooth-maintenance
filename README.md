# smooth-maintenance
The smoothest maintenance mode plugin.

### Maintenance Mode / Coming Soon

**Niche:** Displaying a maintenance or coming-soon page while a site is being built or updated, with subscriber capture, countdown timers, and access control.

**Why this early:** It is the fastest possible ship — perhaps a single week — and generates meaningful install count that builds plugin.org profile and credibility for every plugin that follows.

**Existing Competitors:**

- **SeedProd** — 1,000,000+ installs. Has metastasized into a full landing page builder at $79–$399/year. Coming-soon is just one of dozens of modes. Overkill for most users and expensive.
- **LightStart – Maintenance Mode** (WebFactory) — 700,000+ installs, free with a $39/year Pro. Lean and fast but UI is dated and Gutenberg integration is superficial.
- **Maintenance** (fruitfulcode) — 200,000+ installs, free only. Popular but abandoned-feeling, last significant update years ago.

**What to build to win:**

A Gutenberg-native maintenance plugin where the coming-soon page is built entirely using native WordPress blocks — any Gutenberg block, any FSE pattern, any block-based page builder — without a proprietary drag-and-drop interface. Currently every competitor either builds a custom builder that sits outside WordPress’s editor or provides only basic template options. The developer-friendly angle is equally important: granular access bypass by user role, IP range, or a shareable secret-token preview URL that clients can use to review the site without logging in. These features are paywalled in LightStart and completely absent in the free options.

**Pricing:**

- Free: Full maintenance mode functionality, one design template
- Pro (1 site): $29/year
- Agency (unlimited): $59/year

**SLC v1 — Simple, Lovable, Complete:**

Enable maintenance mode with a single toggle. Choose from five beautifully designed full-screen templates (minimal, bold, countdown, newsletter capture, video background). Any logged-in admin bypasses automatically. Generate a shareable preview link with a secret token. Set a countdown timer. Capture subscriber emails to a built-in list or to Mailchimp/ConvertKit via webhook.

**Ultimate Vision:**

Deep WordPress FSE integration where any Site Editor template can be used as the maintenance page. Smart maintenance windows that automatically enable/disable based on a scheduled deployment. A/B testing of coming-soon designs to optimize subscriber capture rate. Analytics showing visitor behavior during maintenance periods. WooCommerce-aware mode that keeps the shop visible while only locking down specific pages.
